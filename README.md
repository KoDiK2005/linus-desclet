# Vibe Clock — Cinnamon Desklet

Один десклет часов и даты для Cinnamon (Linux Mint) с двумя переключаемыми стилями оформления: **Аниме** и **Persona 5**.

## Возможности

- Переключение стиля («Аниме» / «Persona 5») прямо в настройках десклета — без перезагрузки Cinnamon и без переустановки
- Часы (12/24ч, с секундами или без) и дата на русском языке (полная — в аниме-стиле, сокращённая caps — в Persona 5)
- **Аниме**: chibi-спрайт персонажа (SVG), неоновая розово-фиолетовая палитра, мягкий вертикальный градиент фона
- **Persona 5**: диагональная рваная красно-чёрная графика фона, полутоновые точки, стилизованная маска, жирный курсивный шрифт с резкой тенью
- У каждого стиля — свои настраиваемые цвета, размер персонажа/маски, размер шрифта часов и тексты-декор сверху/снизу

Все настройки доступны через правый клик по десклету → «Настроить десклет».

## Установка

```bash
git clone https://github.com/<your-user>/linus-desclet.git
ln -s "$(pwd)/linus-desclet/cinnamon-desklet/vibe-clock@vibe" ~/.local/share/cinnamon/desklets/vibe-clock@vibe
cinnamon --replace &
```

Затем: правый клик на рабочем столе → «Добавить десклеты» → **Vibe Clock**.

## Структура

```
cinnamon-desklet/vibe-clock@vibe/
├── metadata.json          # метаданные десклета
├── desklet.js              # логика: часы, дата, сборка UI под текущий стиль
├── stylesheet.css           # геометрия для обоих стилей (.anime-clock-box / .p5-clock-box)
├── settings-schema.json     # переключатель стиля + настройки для каждого из них
├── sprite.svg                # chibi-спрайт персонажа (аниме-стиль)
├── background.svg            # диагональная рваная графика фона (Persona 5)
└── mask.svg                  # стилизованная маска (Persona 5)
```

## Как это работает

`desklet.js` хранит текущий выбранный стиль и при его смене полностью пересобирает
поддерево виджетов (`_buildStyleUI`), после чего применяет цвета/размеры/тексты
конкретного стиля (`_applyAnimeTheme` / `_applyPersonaTheme`). Форматирование даты
и времени общее для обоих стилей, но использует разные списки дней недели/месяцев
(полные для аниме, сокращённые для Persona 5, чтобы текст не обрезался на плотном макете).
