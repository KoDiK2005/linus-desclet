# Vibe Clock — Cinnamon Desklet

Один десклет часов и даты для Cinnamon (Linux Mint) с четырьмя переключаемыми
стилями оформления: **Аниме**, **Persona 5**, **Persona 4** и **Persona 3**.

## Возможности

- Переключение стиля прямо в настройках десклета — без перезагрузки Cinnamon и без переустановки
- Часы (12/24ч, с секундами или без) и дата на русском языке
- Настройки каждого стиля показываются только когда он активен (условная видимость по значению «Стиль оформления»)
- Мягкая «дышащая» анимация свечения времени во всех стилях
- **Аниме**: портрет anime-девушки (большие глаза с бликами, twin tails, блеск на волосах, искры), неоновая розово-фиолетовая палитра, вертикальный градиент фона
- **Persona 5**: диагональная рваная красно-чёрная графика с виньеткой, полутоновые точки, маска с акцентной прорезью, жирный курсив с резкой тенью
- **Persona 4**: жёлто-чёрная гамма, ТВ с помехами на экране (Midnight Channel), рваные диагональные полосы, скан-линии
- **Persona 3**: тёмно-синий с золотом, полумесяц, тонкая арт-деко рамка, маска с золотой «слезой»
- У каждого стиля — свои настраиваемые цвета, размер иконки/персонажа, размер шрифта часов и тексты-декор сверху/снизу

Все настройки доступны через правый клик по десклету → «Настроить десклет».

## Установка

```bash
git clone https://github.com/<your-user>/linus-desclet.git
ln -s "$(pwd)/linus-desclet/cinnamon-desklet/vibe-clock@vibe" ~/.local/share/cinnamon/desklets/vibe-clock@vibe
cinnamon --replace &
```

Затем: правый клик на рабочем столе → «Добавить десклеты» → **Vibe Clock**.

## Структура

```
cinnamon-desklet/vibe-clock@vibe/
├── metadata.json          # метаданные десклета
├── desklet.js              # логика: реестр стилей STYLES, сборка UI, часы/дата
├── stylesheet.css           # геометрия и типографика для всех стилей
├── settings-schema.json     # переключатель стиля + настройки для каждого из них
├── sprite.svg                # портрет anime-девушки
├── background.svg / mask.svg      # Persona 5: фон и маска
├── background4.svg / mask4.svg    # Persona 4: фон и телевизор
└── background3.svg / mask3.svg    # Persona 3: фон и маска
```

## Как это работает

Все стили описаны декларативно в объекте `STYLES` в `desklet.js` (CSS-классы,
файл иконки, форматы даты/дней недели, генерируется ли фон или используется
готовое SVG-изображение). Добавление нового стиля не требует правки логики
сборки/применения темы — только добавление записи в `STYLES`, набора настроек
в `settings-schema.json` (с `"dependency": "style=<имя>"`) и CSS-класса.

При смене стиля `desklet.js` полностью пересобирает поддерево виджетов
(`_buildStyleUI`), затем применяет цвета/размеры/тексты (`_applyTheme`).
Для генерируемого фона (аниме) используются настройки цвета фона/свечения;
для стилей с готовым SVG-фоном (Persona 3/4/5) акцентный цвет тонирует
только текст и обводку иконки, а сама фоновая графика фиксирована под тематику.
